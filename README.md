# TDD String Calculator

A robust, test-driven implementation of the String Calculator Kata. This project demonstrates clean code practices, modular JavaScript, and automated CI/CD deployment to Vercel.

## Description
This application parses strings of numbers with customizable delimiters and calculates their sum, adhering to strict TDD rules.

## Features
- **TDD Driven**: Built with Vitest for 100% logic coverage.
- **Custom Delimiters**: Supports multiple and multi-character delimiters.
- **Auto-Deployment**: Integrated GitHub Actions for Vercel.
- **Professional UI**: Clean, developer-focused interface.

## Tech Stack
`javascript` `tdd` `vitest` `vite` `vercel` `github-actions`

## Usage
1. `npm install`
2. `npm test` - Run test suite
3. `npm run dev` - Launch calculator
