Kata Progression (Steps 1-9)

1

Empty String

"" returns 0

2

Single Number

"1" returns 1

3

Two Numbers

"1,2" returns 3

4

Unknown Amount

"1,2,3,4..." handles any count.

5

New Line Delimiter

"1\n2,3" returns 6

6

Custom Delimiters

"//;\n1;2" returns 3

7

Negative Numbers

Thorws negatives not allowed: -x

8

Ignore > 1000

"2,1001" returns 2

9

Multiple/Any Length

"//[***]\n1***2" or "//[*][%]\n1*2%3"
